Custom Scrollbar Example

Note: The -webkit-scrollbar is not supported in Firefox or in Edge, prior version 79. use scrollbar-width in firefox

you can change scrollbar style in chrome 
      /* width */
      ::-webkit-scrollbar {
        width: 20px;
        background-color: lightblue;
      }
      ::-webkit-scrollbar:hover {
        width: 50px;
        background-color: lightblue;
      }

      /* Track */
      ::-webkit-scrollbar-track {
        box-shadow: inset 0 0 5px blue;
        border-radius: 20px;
      }

      ::-webkit-scrollbar-track:hover {
        box-shadow: inset 0 0 5px green;
        border-radius: 20px;
      }
      /* Handle */
      ::-webkit-scrollbar-thumb {
        background: gray;
        border-radius: 5px;
      }

      /* Handle on hover */
      ::-webkit-scrollbar-thumb:hover {
        background: black;
      }
